# 指令三

## 作用

- 一般在当月最后一周执行完指令二之后执行
- 根据当月的周报和月计划执行结果生成月报
- 更新当月月计划的完成情况
- 同步更新 `02_state/current_context.md`

## 执行时优先参考

1. 当月所有周报
2. 当前月计划或阶段计划
3. `02_state/task_index.md`
4. `02_state/current_context.md`

## 预期输出

- 一份新的月报或阶段复盘
- 一份更新后的月计划或阶段计划
- 更新后的动态入口文件

## 执行原则

- 月报重点总结阶段性结果、未完成事项和后续方向
- 月计划完成情况应按真实结果更新，不为了“好看”而虚写完成
- 如果进入新阶段，应在动态入口中切换当前阶段、当前月计划和当前周计划指针
