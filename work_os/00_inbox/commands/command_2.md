# 指令二

## 作用

- 一般在当周最后一个工作日执行完指令一之后执行
- 根据当周日报和当周周计划执行结果生成周报
- 更新当周周计划的完成情况
- 如果不是当月最后一周，则更新下周周计划
- 同步更新 `02_state/current_context.md`

## 执行时优先参考

1. 当周所有日报
2. 当周周计划
3. `02_state/task_index.md`
4. `02_state/current_context.md`

## 预期输出

- 一份新的周报
- 一份更新后的当周周计划
- 如果需要，一份更新后的下周周计划
- 更新后的动态入口文件

## 执行原则

- 周报应以真实执行结果为主，不按原计划硬写完成
- 需要明确区分：
  - 已完成事项
  - 已启动但未完成事项
  - 遗留问题和阻塞
- 如果下周计划需要调整，应直接根据本周结果修正，而不是照搬旧版本
