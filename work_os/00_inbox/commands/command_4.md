# 指令四

## 作用

- 生成新的月计划或阶段计划
- 生成对应周计划
- 生成第一周的日计划
- 初始化 `02_state/current_context.md`

## 适用场景

- 新的月份或新的阶段刚开始时
- 需要从上层计划重新启动一轮完整规划时

## 执行时优先参考

1. `01_context/role.md`
2. `01_context/projects/project_main.md`
3. `02_state/task_index.md`
4. 上一阶段的月报、周报或关键日志
5. `02_state/current_context.md`

## 预期输出

- 一份新的月计划或阶段计划
- 对应的周计划
- 第一周的日计划
- 更新后的动态入口文件

## 执行原则

- 计划优先按工作日划分，不按自然周机械切分
- 第一轮计划应尽量轻量，先保证能执行
- 第一周日计划要从真实重点事项里长出来，而不是平均分配
