# 指令一

## 作用

- 用户输入当日的工作结果和内容
- 生成当日日报
- 同步更新当天日计划中的实际完成情况
- 如果当天不是当周最后一个工作日，则同步调整明天的日计划
- 如果任务真实状态发生变化，则同步更新 `02_state/task_index.md`
- 同步更新 `02_state/current_context.md`

## 执行时优先参考

1. 当天的日计划
2. 当前周计划
3. 最近 1 到 3 天的日报
4. `02_state/task_index.md`
5. `02_state/current_context.md`

## 预期输出

- 一份新的或更新后的日报
- 一份更新后的当日日计划
- 如果需要，一份更新后的明日日计划
- 如果需要，更新后的任务总表
- 更新后的动态入口文件

## 执行原则

- 如实记录，不编造进展
- 即使当天产出不多，也可以正常形成日报
- 日计划中的勾选状态、实际完成情况和日报内容应保持一致
- 如果发现当天完成情况会影响明天重点，应直接反映到明天日计划中
