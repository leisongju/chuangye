# 任务总表

## 状态定义

- `todo`：尚未开始
- `doing`：正在推进中
- `blocked`：暂时被卡住，无法继续
- `waiting`：等待他人或外部条件
- `done`：已经完成，无需继续动作
- `dropped`：已决定不再继续

## 优先级定义

- `P0`：紧急且重要，需要立即推进
- `P1`：重要，应在本周安排
- `P2`：有价值但不关键
- `P3`：可选事项，锦上添花

## 更新规则

- 任务真实状态变化时，及时更新 `status`。
- 持续维护 `next_action`，让明天规划时不用从零重新思考。
- 只有在确实被外部条件阻塞时才填写 `blocker`。
- 每次修改任务行时同步更新 `updated_at`。
- 如果任务已完成，将 `status` 设为 `done`，并清空 `blocker`。
